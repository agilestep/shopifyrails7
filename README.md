# Shopify APP on Rails 7

This is an app to test Shopify app creation with Rails 7 

* Ruby version: 3.1.3

* To try it just: 
1. git clone  
2. bundle install
3. create application.yml and fill SHOPIFY_API_KEY and SHOPIFY_API_SECRET
```
   HOST: http://localhost:3000
   SHOPIFY_API_KEY: 
   SHOPIFY_API_SECRET:   
```

4. rails s
